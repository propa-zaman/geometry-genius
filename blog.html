DOM means Document Object Model. It is a programming interface for web document. DOM allow us to manipulate the data in the webpage. it gives us a lot of functionality.

We select html elements using methods of DOM. There are 5 ways to select elements in DOM. They are-
  1. getElementsByTagName()
  2. getElementsByClassName()
  3. getElementById()
  4. querySelector()
  5. querySelectorAll()

Event bubble is a kind of DOM propagation. It means when an element receives an event, this event propagate to the element's parent. It keeps propagating until reaching the root element.

Callback function is function, which is invoked into another function as an argument. When we have to wait for a function to be execute first, then we have to use a callback function under it . Because, we have to wait for previous function call first. Then we have to call another function.
