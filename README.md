# Geometry Genius

Geometry Genius is a web application that allows users to calculate the areas of different geometric shapes and generate random colors for each shape.

## Live website: https://relaxed-panda-45beda.netlify.app/

## Features

- Triangle Area Calculation: Calculates the area of a triangle based on the input base and height values.
- Rectangle Area Calculation: Calculates the area of a rectangle based on the input width and length values.
- Parallelogram Area Calculation: Displays a fixed area value for the parallelogram.
- Rhombus Area Calculation: Displays a fixed area value for the rhombus.
- Pentagon Area Calculation: Displays a fixed area value for the pentagon.
- Ellipse Area Calculation: Displays a fixed area value for the ellipse.
- Random Color Generation: Generates a random background color for each shape when hovering over the shape.

## Tech Stack

This application is built with HTML, CSS, and Vanilla JavaScript.
